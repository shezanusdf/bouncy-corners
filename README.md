# Bouncing DVD Logos

A smooth, flicker-free Streamlit app that simulates multiple bouncing DVD logos.

## Features

- Multiple logos with random colors
- Adjustable animation speed
- Corner bounce counter
- Smooth playback even at high frame rates

## How to Run

1. Clone the repository:

```bash
git clone <repo-url>
cd <repo-directory>
```
2. Install dependencies:
```bash
pip install -r requirements.txt
```
3. Run the app:
```bash
streamlit run app.py
```
